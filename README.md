### 🚀 Projet mis en avant

#### 📊 [GA4 E-commerce Analytics — BigQuery × Power BI](https://github.com/CarlFlereau-Data/ga4-analytics-bigquery-powerbi)

Analyse complète du comportement d'achat sur le **Google Merchandise Store**, à partir du dataset public GA4 dans BigQuery.

- 🗂️ Modèle en étoile : **8 vues SQL**, 3 faits + 5 dimensions, 9 relations N:1
- 🧮 **30+ mesures DAX** : conversion, MoM, moyennes mobiles, rétention de cohorte, LTV
- 📈 **5 pages de dashboard** : Overview, Acquisition, Behavior, Funnel, Cohort/LTV
- 🐛 Résolution de pièges réels : filtre `debug_mode`, `ANY_VALUE`/`IGNORE NULLS`, déduplication du revenu

`SQL` · `BigQuery` · `Power BI` · `DAX` · `Snowflake` · `Google Analytics`
